# Space Scurvy
Space Scurvy is a 2d simple platformer game made in rust using the macroquad library. This is my second ever game and the first one with any resemblence of scale, and i have learnt a lot from it! I hope at least... 

The game features a pirate... in space!! I know revolutionary. Throughout the game you will encounter many foes from fish to jetpacker. This is my second ever game and the first one with any resemblence of scale, and i have learnt a lot from it! I hope at least...
